# 🔋 Smart Battery Health Predictor

A Machine Learning web application that predicts smartphone battery health based on user behavior and device statistics.

## Features

- Predicts Battery Health
- Gradient Boosting Classifier
- Interactive Streamlit Web App
- Personalized Maintenance Advice
- Confidence Score
- Battery Health Score

## Technologies Used

- Python
- Streamlit
- Pandas
- Scikit-Learn

## Dataset

- Smartphone Battery Health Dataset
- 5000 Records

## Machine Learning Models Compared

- Logistic Regression
- Decision Tree
- SVM
- Random Forest
- Gradient Boosting

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
